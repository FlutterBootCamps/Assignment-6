## Password Reset Pages (3 Pages)
In these pages the user will go through the password reset process, by entering their email, then getting an OTP and typing it, then being able to change thier password, also throughout any input action, no empty or invalid/incorrect input allowed, the OTP only accepts numbers to be typed and password has to be 6 characters or higher.
